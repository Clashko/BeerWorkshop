1. Создать папку для проекта BeerWorkshop.Client и вней создать проект на основе Vite React Typescript.
2. Добавить в проект VitePWA плагин и настроить его. Для картинок просто вставь шаблонные имена - я позже их создам.
3. Добавить в проект tailwind и настроить его.
4. Добавить в проект MaterialTailwind библиотеку и настроить её.
5. Добавить в проект Redux toolkit и при необходимости настроить.
6. Для роутинг добавить в проект React-router dom.
7. Для форм добавить reat-hook-forms и для валидации yup.
8. Для иконок добавить в проект react-icons.
9. Создать компонент ThemeManager - для возможности изменения темы приложения (темы: светлая и темная). Данный компонент должен содержать кнопку с иконкой в виде лампочки по нажатию на которую будет происходить изменение темы приложения.
